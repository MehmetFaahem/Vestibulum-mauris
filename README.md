A responsive website built with HTML, CSS, and JavaScript featuring Japanese content and modern design.

## Features

- **Responsive Design**: Optimized for desktop, tablet, and mobile devices with 768px breakpoint
- **Modern Layout**: Clean, professional design with smooth animations
- **Mobile Navigation**: Hamburger menu for mobile devices
- **Smooth Scrolling**: Navigation with smooth scroll to sections
- **Interactive Elements**: Hover effects and animations
- **Japanese Content**: Properly formatted Japanese text content

## File Structure

```
├── index.html          # Main HTML file
├── styles.css          # CSS styles with responsive design
├── script.js           # JavaScript functionality
├── images/             # Image assets
│   ├── mainImg.jpg     # Hero section image
│   ├── reason.jpg      # Reason section image
│   ├── bgSec04.jpg     # Success section background
│   ├── sec02_01.jpg    # Story section image 1
│   ├── sec02_02.jpg    # Story section image 2
│   ├── circleImg01.png # Service icon 1
│   ├── circleImg02.png # Service icon 2
│   └── circleImg03.png # Service icon 3
└── texts/              # Original text content (RTF files)
```

## Sections

1. **Header**: Fixed navigation with company logo
2. **Hero**: Main visual with company tagline
3. **Message**: Three message blocks with company information
4. **Reason**: Why choose us section with image and text
5. **Service**: Three service offerings with icons
6. **Story**: Company story with images
7. **Success**: Success stories with background image
8. **Vision**: Company vision and goals
9. **Company**: Company information and contact details
10. **Footer**: Copyright information

## Responsive Breakpoints

- **Desktop**: 1200px and above
- **Tablet**: 768px - 1199px
- **Mobile**: Below 768px

## Browser Support

- Modern browsers (Chrome, Firefox, Safari, Edge)
- Mobile browsers (iOS Safari, Chrome Mobile)

## Usage

1. Open `index.html` in a web browser
2. Navigate through sections using the header menu
3. On mobile devices, use the hamburger menu for navigation

## Technologies Used

- HTML5
- CSS3 (Grid, Flexbox, Animations)
- Vanilla JavaScript (ES6+)
- Responsive Web Design
- Japanese Typography Support
